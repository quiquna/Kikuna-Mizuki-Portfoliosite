ポートフォリオサイト

Type: Webデザイン / ノーコードツールによる制作

就職活動にあたり、自身の経歴や制作物をまとめたポートフォリオサイトを制作しました。必要最低限のページ構成でシンプルにまとめつつも、自身の強みを伝えられるサイトを目指しました。

ターゲット
Web制作会社の採用担当者の方々
目的
就職活動を行うにあたり、自身の経歴や強み、またこれまでの制作物や思考を伝えるため
情報設計
採用担当者の方がどういう流れで見るかを考え、ページ構成を組みました。 まずTOPで全体の概要を知っていただいた後、Aboutページでは、なぜデザイナーを目指そうと思ったのか、どのような部分で貢献できるのか、根拠を持って伝えています。そのように自分についての情報を伝えた上で、制作物もあわせて見ていただき、自身の人となりとスキルをセットで伝えられるよう構成しました。
レイアウト
過去に作成したものを見てもらうことが1番の目的であるため、余白を多めに取り、シンプルなレイアウトで構成しています。
デザイン
全体的にテキスト量が多くなってしまうため、青と白を基調としたシンプルな配色を用いて、長く見ていても疲れないよう工夫しています。 また、ゴシック体を使用することで期待感や、成長といった力強い印象を与えることを意図しています。サイトを開いた瞬間に前向きな印象でインパクトを与えたいと思い、TOPページのファーストビューは特にフォントを太めに設定しています。
製作期間
ワイヤーフレーム：３日
デザイン：１週間
コーディング：５週間
使用ツール
Illustrator / Photoshop / Visual Studio Code
